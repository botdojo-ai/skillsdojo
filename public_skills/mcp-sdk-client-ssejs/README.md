# mcp-sdk-client-ssejs

Client transport alternative of @modelcontextprotocol/sdk/client base on sse.js. The main purpose is make it working on React Native with llama.rn.

## Source

- **Type**: mcp-package
- **Source**: npm
- **Repository**: git+https://github.com/mybigday/mcp-sdk-client-ssejs.git
- **npm**: https://www.npmjs.com/package/mcp-sdk-client-ssejs



## Installation

```bash
# Clone the repository
git clone git+https://github.com/mybigday/mcp-sdk-client-ssejs.git
```

```bash
# Install from npm
npm install mcp-sdk-client-ssejs
```
