# puppeteer-mcp-server

Experimental MCP server for browser automation using Puppeteer (inspired by @modelcontextprotocol/server-puppeteer)

## Source

- **Type**: mcp-package
- **Source**: npm
- **Repository**: git+https://github.com/merajmehrabi/puppeteer-mcp-server.git
- **npm**: https://www.npmjs.com/package/puppeteer-mcp-server



## Installation

```bash
# Clone the repository
git clone git+https://github.com/merajmehrabi/puppeteer-mcp-server.git
```

```bash
# Install from npm
npm install puppeteer-mcp-server
```
